Pemrograman Web

Website ini didukung dan diajarkan oleh Mata Kuliah Pemrograman Web yang diberikan oleh Bapak Riswan S. Sianturi selaku dosen Pemrograman Web TIF A. Disini saya akan menjabarkan dua aspek atau dua bahasa yang digunakan pada Website ini, yaitu HTML dan CSS.

HTML (Hyper Text Markup Language)

Hypertext Markup Language atau HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman website. HTML terdiri dari kombinasi teks dan simbol yang disimpan dalam sebuah file. Dalam membuat file HTML, terdapat standar atau format khusus yang harus diikuti. Format tersebut telah tertuang dalam standar kode internasional atau ASCII. Tak disangka, kini HTML telah berevolusi menjadi bahasa markup yang telah digunakan oleh lebih dari 92,3% persen website di dunia. Tim Berners-Lee merupakan sosok ilmuwan dibalik lahirnya HTML pada tahun 1991.

CSS (Cascading Style Sheets)

SS adalah bahasa Cascading Style Sheet dan biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML. CSS berfungsi untuk memisahkan konten dari tampilan visualnya di situs.CSS dibuat dan dikembangkan oleh W3C (World Wide Web Consortium) pada tahun 1996 untuk alasan yang sederhana, karena dulu HTML tidak dilengkapi dengan tags yang berfungsi untuk memformat halaman. CSS dapat mengatur warna teks, jenis font, baris antar paragraf, ukuran kolom, jenis background yang dipakai, serta untuk mendesain layout.
